# To-do-List

## Təsvir
Html və JavaScript istifadə edilərək hazırlanmış to-do list tətbiqidir. To-do-ları əlavə etmək, silmək, axtarmaq, filtrləmək üçün istifadə olunur.To-dolar Local Storage-də saxlanılır

## Funksiyalar
- Yeni to-do əlavə etmək
- To-do silmək
- Bütün to-do-ları bir düymə ilə silmək
- To-doları axtarmaq, filtr etmək
- Səhifə yeniləndikdə yaddaşda saxlanmış to-doları göstərmək
- To-do-ları local storage-də saxlamaq

## Texnologiyalar
- HTML
- Bootstrap
- JavaScript
- Local Storage

## Komanda işi
To-do list layihəsi iki nəfər tərəfindən komanda şəklində Github-dan istifadə edilərək hazırlanmışdır
